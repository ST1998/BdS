- Introduction -

It is the first time that I write an article about one of my childhood games. Re-Volt is an atypical Racing Game, with radio-controlled vehicles that run on tracks that are reproductions of various real-life (and not) environments, which, from the point of view of vehicles, appear gigantic, which gives an unusual appearance to the tracks (that's why I called it atypical).

RVGL is a project carried out by the community where there is the Re-Volt game completely rewritten to guarantee the complete native support on Windows (from 7 up to 10), GNU / Linux and Android (from 2.x onwards). Being Open-Source, anyone can get their hands on the source code and expand it with new vehicles, new tracks and new features that weren't in the original game.

In this article, it will be dedicated in the installation of Re-Volt in Android devices and the nice thing is that it can run in obsolete Android versions (except 1.x). So, if you want to play it on a bad Tablet or on a Smartphone older than your shoes, you can do it! And you can play it in portrait and landscape mode! It does not require any root permission and there is no risk of breaking the device. Let's start!

- Setup -

( 1 ) Create a folder called Re-Volt and put in that folder all the files you previously downloaded, as you can see in this image:



( 2 ) If you downloaded files on an Android device, skip the step, otherwise, from Android activate the USB Debug from the Developer Options, and then connect the Smartphone to the PC (if you downloaded the files from there) via the USB cable. Once this is done, from the status bar (located at the top), scroll down to open the curtain and press File transfer via USB activated and select File transfer. Once this is done, the device you have connected will appear in the File Manager on PC and press there, then enter the folder where it says Internal memory and finally copy / paste the folder Re-Volt.

( 3 ) From the File Manager on Android and from the Re-Volt folder, press rvgl_20.0430a_android.apk (20.0430a indicates the version of the game) and, in case you get the warning that you are about to install an app that is not from the Google Play Store, don't worry! It does not contain any malware. In order to install it, you will need to activate the possibility to install apps from unknown sources on Android.

( 4 ) Once the game is installed, if you want to play with a Wireless Controller, press rvgl_controller_map.apk. Once installed, open and if you get the warning that says "Configure controller 0: Android Accelerometer"? press Yes and, from your Controller, configure each button and lever that the app highlights you in green. Once finished, the setup will be complete and the app will close. I advise you not to uninstall that app.

( 5 ) If you want to play with the Soundtrack and the additional contents of the Dreamcast version, from the Re-Volt folder, extract soundtrack.zip (or rv_soundtrack_dc_flac.7z) and rvgl_dcpack.zip. Next, move everything you extracted from the Re-Volt folder to the RVGL folder (located in the internal storage). If it asks you to overwrite the files, confirm the request!

( 6 ) From the Android Home Screen, run RVGL (indicated with the lightning bolt icon) and if the game works completely, then you have performed the whole procedure correctly! Have fun!
